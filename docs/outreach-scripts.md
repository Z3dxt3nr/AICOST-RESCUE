1. What AI tools do you currently pay for?
2. Rough monthly spend?
3. Is the spend mostly subscriptions, API tokens, or both?
4. What work do you use AI for most?
5. What surprised/frustrated you about the cost?
6. Do you track model usage by workload?
7. Would a monthly cost report or alert be useful?
8. If we found AUD $300+/month in savings, would a paid audit be worth it?
