Leon chose two first audiences:

1. Dev teams using AI coding tools and APIs.
2. Agencies/consultants using AI to deliver client work.

Start with warm/manual outreach. Do not mass spam. The first goal is 10 conversations and 1-3 paid founding audits.

AICost Rescue is a 48-hour AI spend audit for teams using tools like Cursor, GitHub Copilot, Claude Code, OpenAI, Anthropic, Gemini, OpenRouter, and agent workflows.

It produces:

- likely cost leaks

- quick wins
- estimated monthly savings
- 30-day implementation plan

## Reddit / community version

Use this only when it genuinely fits the thread. Give value first.

```text
One thing I’d watch is AI tool creep.

A lot of teams start with one or two tools, then suddenly have Cursor, Copilot, ChatGPT Team, Claude, API usage, automation tools, and no clear idea which spend is actually producing value.

A simple check:
- list every AI subscription/API bill,
- estimate spend by workload,
- separate low-risk tasks from high-risk tasks,
- move low-risk repetitive work to cheaper models,
- add caching for repeated prompts/documents,
- cut duplicated seats/tools.

I’m validating a small AI spend audit around this. If anyone has rough monthly AI spend + tools used, I’m happy to give a free waste-band estimate while I’m testing the process.
```

## Discovery questions

1. What AI tools do you currently pay for?
2. Rough monthly spend?
3. Is the spend mostly subscriptions, API tokens, or both?
4. Team size?
5. What work do you use AI for most?
6. What surprised/frustrated you about the cost?
7. Do you track model usage by workload?
8. Do you use AI in client delivery or internal work?
9. Would a monthly cost report or alert be useful?
10. If we found AUD $300+/month in savings, would a AUD $499 audit be worth it?
