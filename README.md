# AICost Rescue

A service-first MVP for an AI Spend Audit + Optimizer.

## Offer

Help dev teams, agencies, founders, and SMBs cut AI tool/API costs across Cursor, Copilot, Claude Code, OpenAI, Anthropic, Gemini, OpenRouter and agent workflows.

## MVP features

- Landing page
- Interactive AI cost self-check calculator
- Mailto-based audit request CTA
- Stripe checkout for Team / Agency Audit
- Report template
- Outreach scripts
- Tested calculator logic

## Local commands

```bash
npm install
npm run dev
npm run build
npx vitest run
```
